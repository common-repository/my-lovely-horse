== Description ==

My Lovely Horse is a very simple re-write of the ‘Hello Dolly’ plugin, so that it now displays the lyrics to the song ‘My Lovely Horse’, from the hit British TV comedy ‘Father Ted’.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/my_lovely_horse` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. That's it, you're done.
